pwd → Print Work Directory

Print in your cmd your absolut directory cd → Change Directory

Change the current directory by placing (or not) the name of the directory.

Examples:
1st -------TERMINAL--------------------------------
cd workspace
-------TERMINAL--------------------------------

•This one will enter in the 'workspace' directory

2nd -------TERMINAL--------------------------------
cd ../
-------TERMINAL--------------------------------

•This one will leave the current directory

3rd -------TERMINAL--------------------------------
cd ~
-------TERMINAL--------------------------------

•This one will go back to the very first main directory

 ls → list

List the archives in the current directory Examples:

  • -l → more info
  • -a → all archives
  • -h → human friendly
  • -R → Recursive order
  • -S → Size order
 mkdir → Make Directory
Make a directory from scrach, being able to create one inside another
-------TERMINAL--------------------------------
mkDir Morpheus
-------TERMINAL--------------------------------

• This command will create a directory named 'Morpheus'

-------TERMINAL--------------------------------
cd -p Morpheus/Matrix
-------TERMINAL--------------------------------

• This one with -p (parent) will create a directory inside 'Matrix' named 'Morpheus'
